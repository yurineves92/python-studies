## Módulo 1: Fundamentos Básicos

**Duração:** 2-3 semanas  
**Nível:** Iniciante

### Conteúdo

#### 1. Sintaxe Básica

**1.1 Variáveis e Tipos de Dados**

Aprenda sobre:
- Declaração de variáveis
- Tipos primitivos: int, float, string, bool
- Conversão de tipos
- Função type()

Exemplo prático:
- Criar variáveis de diferentes tipos
- Imprimir valores e tipos
- Usar formatação de strings

**1.2 Operadores**

Aprenda sobre:
- Operadores aritméticos: soma, subtração, multiplicação, divisão, divisão inteira, resto, potência
- Operadores de comparação: igual, diferente, maior, menor, maior ou igual, menor ou igual
- Operadores lógicos: and, or, not

Exemplo prático:
- Realizar operações matemáticas
- Comparar valores
- Usar lógica booleana

**1.3 Input e Output**

Aprenda sobre:
- Função print()
- Função input()
- Formatação com f-strings

Exemplo prático:
- Receber entrada do usuário
- Processar dados
- Exibir resultados formatados

---

#### 2. Estruturas de Controle

**2.1 Condicionais**

Aprenda sobre:
- Estrutura if, elif, else
- Condições aninhadas
- Operadores de comparação em condicionais

Exemplo prático:
- Verificar maioridade
- Criar calculadora simples com operadores

**2.2 Loops**

Aprenda sobre:
- Loop for com range()
- Loop while
- Break e continue
- Pass

Exemplo prático:
- Contagem progressiva e regressiva
- Iterar com condições
- Controlar fluxo de loops

---

#### 3. Estruturas de Dados Básicas

**3.1 Listas**

Aprenda sobre:
- Criar e manipular listas
- Métodos: append(), insert(), remove(), pop()
- Acessar elementos por índice
- Slicing (fatiamento)
- Iterar sobre listas

Exemplo prático:
- Lista de frutas
- Adicionar e remover elementos
- Acessar primeiro e último elemento
- Fatiar listas

**3.2 Tuplas**

Aprenda sobre:
- Imutabilidade das tuplas
- Desempacotamento
- Quando usar tuplas

Exemplo prático:
- Coordenadas x, y
- Dados de pessoa
- Desempacotar valores

**3.3 Dicionários**

Aprenda sobre:
- Pares chave-valor
- Acessar valores por chave
- Adicionar e modificar elementos
- Métodos: keys(), values(), items()
- Iterar sobre dicionários

Exemplo prático:
- Dados de aluno com nome, idade, curso, notas
- Adicionar email
- Iterar e exibir informações

**3.4 Conjuntos (Sets)**

Aprenda sobre:
- Conjuntos sem duplicatas
- Adicionar elementos
- Operações: união, interseção, diferença

Exemplo prático:
- Criar conjuntos de números
- Operações matemáticas de conjuntos

**3.5 Strings**

Aprenda sobre:
- Métodos: upper(), lower(), title(), replace()
- Split e join
- Verificações: startswith(), in
- Formatação

Exemplo prático:
- Manipular texto
- Dividir em palavras
- Verificar conteúdo

---

### Objetivos de Aprendizagem - Módulo 1

Ao final deste módulo, você será capaz de:
- Entender a sintaxe básica do Python
- Trabalhar com diferentes tipos de dados
- Usar estruturas de controle de fluxo
- Manipular estruturas de dados básicas

**6.2 Arquivos CSV**

Exemplo prático:
- Criar arquivo CSV com dados de pessoas
- Ler e exibir dados do CSV

**6.3 Arquivos JSON**

Aprenda sobre:
- Módulo json

- json.load() para ler

Exemplo prático:
- Salvar dados de pessoa em JSON
- Ler e processar JSON

Exemplo prático:
- Tratar entrada inválida

- Tratar arquivo não encontrado

**7.2 Exceções Customizadas**

Aprenda sobre:
- Criar classe de exceção
- Raise para lançar exceção
- Capturar exceção customizada

Exemplo prático:
- Criar IdadeInvalidaError
- Validar idade
- Tratar erro customizado
